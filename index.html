Home Watcher

Currently in development

Web application created to maintain surveillance of a location using a Raspberry Pi camera module. The Raspberry Pi camera is triggered using a motion sensor and the picture is sent to Amazon Web Services for storage. A client can log in and see the pictures and what time the motion was recorded.
